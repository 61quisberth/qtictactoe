Dependencies

Qt 5, open-source installer suggested, http://www.qt.io/download-open-source/
C++11-enabled compiler

Building
---
Out-of-source build is encouraged, to do so:
- mkdir build
- cd build
- qmake ../tictactoe.pro
- make
